Voici un petit mémo :-)

Mémo du Javascript :

\n permet d'effectuer un retour à la ligne dans une chaîne de caractères

Concaténation :
+ . Ex: "Bon"+"jour" donne "Bonjour"

Les commentaires

Pour exclure certaines lignes de l'exécution on les préfixe de //.
On les transforme donc en commentaires.

Les commentaires en Javascript:
// Comentaire sur une ligne.
/* */ Commentaires sur plusieurs lignes.

Les variables

On déclare une variable à l'aide du mot-clé var

var prenom = prompt("Entrez votre prénom :");
alert("Bonjour, " + prenom);

l'ordre prompt affiche une boîte contenant le message entre parenthèses
l'ordre alert affiche une seconde boîte contenant le résultat de la concaténation de la chaîne "Bonjour"
avec la valeur de la varable "prenom".

Déboguage :
dans firefox -> ouvrir le fichier html qui renvoie au fichier js -> alt -> outils ->
développement web -> outils de développement -> débogueur -> console scindée.

Le débogueur permet d'afficher le ésultat de la commande console.log

Les conditions

var nombre = Number(prompt("Entrez un nombre :"));
if (nombre > 0) { console.log(nombre + " est positif");
}
On appelle ce qui se trouve entre accolades un bloc de saisie associé à l'instruction if.
Pour créer des expressions booléennes (vrai ou faux) on utilise des opérateurs de couparaisons :
=== Egal à
!== Différent de
< Inférieur à
>= Inférieur ou égal à
> Supérieur à
>= Supérieur ou égal à

L'opérateur logique ET : &&
L'opérateur logique OU : ||
